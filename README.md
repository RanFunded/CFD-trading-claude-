# Claude CFD Trading Assistant

A starter repository for researching and monitoring CFD (Contract for Difference) markets using Anthropic Claude APIs.

## Features
- Market data ingestion layer (placeholder)
- Signal generation framework
- Risk management module
- Claude-powered market commentary
- Backtesting structure

## Disclaimer
This project is for educational and research purposes only. It does not provide financial advice and should not be used to make trading decisions without independent validation.

## Quick Start

```bash
git clone <your-repo>
cd claude-cfd-trading-repo
pip install -r requirements.txt
python src/main.py
```

## Architecture

- `src/main.py` – entry point
- `src/claude_client.py` – Claude integration
- `src/strategy.py` – strategy logic
- `src/risk.py` – risk controls
- `tests/` – unit tests
